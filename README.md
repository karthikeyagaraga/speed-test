# WiFi Speed Test Website

A simple, beautiful web application to test your internet speed. Check your download speed, upload speed, and ping latency in just one click!

## Features

- **Download Speed Testing** - Measures how fast you can download data
- **Upload Speed Testing** - Measures how fast you can upload data
- **Ping/Latency Measurement** - Shows your network latency
- **Real-time Progress Tracking** - Visual feedback during the test
- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Modern UI** - Beautiful gradient design with smooth animations

## How to Use

1. Open `index.html` in your web browser
2. Click the "Start Speed Test" button
3. Wait for the test to complete (approximately 15 seconds)
4. View your results in the metric displays

## Files

- `index.html` - HTML structure and layout
- `styles.css` - Styling and responsive design
- `script.js` - Speed testing logic

## Technical Details

- **Frontend Only** - No server required, runs entirely in the browser
- **Client-side Testing** - Tests are simulated based on browser fetch API performance
- **No Dependencies** - Pure HTML, CSS, and JavaScript

## Browser Support

- Chrome/Chromium (recommended)
- Firefox
- Safari
- Edge
- Any modern browser with ES6 support

## Notes

- The test duration is approximately 15 seconds
- Results are for local measurement purposes
- Actual speed may vary based on your network conditions
- For most accurate results, close other applications and ensure no other downloads are running
